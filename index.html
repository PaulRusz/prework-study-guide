Git

  • Refer to 3.1.6 for an example on how to merge branches
  • git pull origin main: makes sure current branch is up to date & syncs/merges code into the GitHub repository
    • git push origin ____: put in branch name where ___ are to push changes
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages changes in the working branch and prepares changed files to be added to the working directory
  • git commit -m "added code to HTML file": commits changes to repository - any comment can go in the ""
